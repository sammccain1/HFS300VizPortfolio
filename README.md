# HFS 300 Final Project Viewing Instructions

## Setup
- Download hfs300mccainfinal.ipynb from GitHub
- Download the datasets from Datasets folder from GitHub

## Colab Instructions
- OpenGoogle Colab at https://colab.research.google.com/
- On the left sidebar, click the Upload tab abd upload hfs300mccainfinal.ipynb from your device to colab
- Once you upon hfs300mccainfinal.ipynb, click the folder icon on the left sidebar
- Select the upload button underneath the word files and upload the datasets 
    1. chicago_neighborhoods.geojson
    2. 2024_Gang_Boundaries.geojson
    3. CPS23-24performancedata.csv
    4. 311_chicago_abandoned_buildings.csv
